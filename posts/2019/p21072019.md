Why:

- free

setup the angular app

```bash
npm install -g @angular/cli
ng new wedding
cd wedding
ng add @angular/material
yarn add firebase @angular/fire
```

setup firebase (during init only select hosting)

```bash
npm i -g firebase-tools
firebase login
firebase init
firebase deploy
```
